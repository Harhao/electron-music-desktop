> 基于electron-vue开发的跨平台QQ音乐PC端，兼容linux,windows,darwin
## 介绍

这个`QQ`音乐`desktop`端主要依赖`electron-vue`和`elementui`端


## 安装使用

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

```
